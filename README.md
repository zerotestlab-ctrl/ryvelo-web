Ryvelo web app (Next.js App Router).

## Getting Started

1) Create an env file:

```bash
copy .env.example .env.local
```

2) Install dependencies (requires npm registry access):

```bash
npm install
```

3) Run the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Auth

- Clerk sign-in: `/sign-in`
- App: `/dashboard`

### Design

- Dark mode default
- Primary navy `#0A2540`
- Accent teal `#00D4C8`
